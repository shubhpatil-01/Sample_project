What is Git?
Git is a distributed version control system that tracks changes in files over time.
It lets you:
Save versions of your work
Go back to earlier state
Collaborate without overwriting each other
Experiment safely using branches
Core Concepts (the stuff that matters)
Repository (repo)
A project tracked by Git.
git init
Commit
A snapshot of your files at a moment in time.
git add .
git commit -m "Meaningful message"
Branch
A parallel line of development (for features, experiments, fixes).
git branch new-feature
git checkout new-feature
# or
git switch -c new-feature
Remote
A copy of your repo stored elsewhere (like GitHub).
git remote add origin URL
git push -u origin main
Everyday Git Workflow
git status        # see what's changed
git add file.py   # stage changes
git commit -m "Describe what changed"
git pull          # get latest changes
